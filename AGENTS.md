# Power Drive Motor ‚Äî Agent Guide

## Commands
```bash
npm run dev       # Frontend dev server (Vite, :5173, proxies /api ‚Üí :3001)
npm run build     # Production build ‚Üí dist/
npm run server    # Express backend (:3001, serves dist/ in prod)
npm run seed      # Populate DB + create admin user (run once)
node server/index.js  # Run everything together (:3001)
```

## Project Structure
- `server/` ‚Äî Express + SQLite backend, API at `/api/`
- `src/pages/` ‚Äî 7 public + 4 admin pages
- `src/data/cars.js` ‚Äî Static fallback (used when API unavailable)
- `src/contexts/CarContext.jsx` ‚Äî Fetches from API, falls back to static `cars.js`
- `src/utils/images.js` ‚Äî `thumbnail(url, size)` for responsive image URLs
- `server/uploads/` ‚Äî Admin-uploaded car images
- `server/data.db` ‚Äî SQLite database (autogenerated by seed)

## Key Facts
- **Admin:** `/admin/login` ‚Äî `admin` / `admin123`
- **Domain:** `powerdrivemotor.com` (GoDaddy, not yet deployed)
- **Server:** port 3001, Vite proxies `/api` and `/uploads` to it
- **Node:** `/tmp/node-v22.14.0-darwin-x64/bin/node` (macOS 12, no system Node)
- **Sharing:** Cloudflare `trycloudflare.com` tunnel (new URL per start)
- **Desktop shortcuts:** `Start Power Drive.command` / `Stop Power Drive.command`

## Data Flow
- **With server:** `CarContext` ‚Üí `GET /api/cars` ‚Üí SQLite ‚Üí admin can CRUD
- **Without server:** `CarContext` falls back to `src/data/cars.js` (read-only)
- **Admin adds car:** POST `/api/cars` ‚Üí file upload ‚Üí saves to SQLite + `uploads/`
- **Contact form:** POST `/api/messages` ‚Üí stored in SQLite, viewed at `/admin/messages`
- **Financing calculator:** pure client-side JS (no backend needed)

## Image Handling
- Dealership images: `cdn-thumbor.autodealersdigital.com/unsafe/fit-in/1000x750/...`
- Cards use `thumbnail(url, "400x300")` to resize via thumbor URL param
- Admin uploads stored at `/uploads/filename` (local filesystem)
- Multiple images per car stored as JSON array in `images` column
- Single `image` column keeps the primary/thumbnail image
- Each car now has **all interior + exterior** images from its detail page (16‚Äì25 per car)

## Admin API Flow
- Cars CRUD at `/api/cars` (requires Bearer JWT token)
- Login at `/api/auth/login` returns JWT (valid 7 days)
- Image upload uses `multipart/form-data` with field name `images` (array, max 10)
- First uploaded image is always the primary thumbnail
- Existing images preserved via `existingImages` JSON field in PUT body

## Gotchas
- Server must be running for admin, contact form, dynamic car data
- Email notifications via Nodemailer + Gmail (requires `EMAIL_PASS` env var = Gmail App Password)
- DB path configurable via `DB_PATH` env var (set `/data/data.db` if using Render Persistent Disk at `/data`)
- `.command` files start server with `nohup` (keeps running after terminal closes)
- Domain parked at GoDaddy ‚Äî not yet pointed to any hosting
- Cloudflare tunnel URL changes every restart (random subdomain)
- User communicates in Arabic using Latin script (e.g., "shu", "bde", "keef")
- Canvas animation was removed (CSS-only effects used instead to avoid lag)
